## 📁 目录结构

```

│   └── 面试准备/      # 面试问题清单、自我介绍
```

## 🎯 使用方法

### 1. 快速捕获
新想法、新资料先放入 `0-Inbox/`，定期整理

### 2. 学习新技术
- 用 `skill-graphify` 解析论文/代码库
- 提取关键概念，用模板创建笔记
- 更新对应领域的 MOC

### 3. 面试准备
- 查看 `1-Projects/面试准备/面试问题清单.md`
- 用 `5-Templates/面试题模板.md` 创建答案

### 4. 知识检索
- 通过 MOC（Map of Content）导航
- Obsidian 内搜索关键词
- 双链 `[[...]]` 跳转相关笔记
